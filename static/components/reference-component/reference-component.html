Perform Actions

process

Define a process

process my_process_name

navigate

Navigate to a url Use state.server to use the server passed in as part of the args

navigate ${state.server}

perform_close_window

Close the window for a given index

perform_close_window 0

perform_refresh

Perform a refresh on the current browser tab

perform_refresh

perform_click

Click on an element

perform_click #element

perform_type_text

Type text into an element

perform_type_text #element "text"

dbl_click

Perform a double click action on a defined element

perform_dbl_click #element

context_click

Perform context click on defined element

perform_context_click #element

click_sequence

perform a sequence of click events over multiple elements

perform_click_sequence ["#element1", "#element2"]

press_key

perform a key press on a defined element

perform_press_key #element ENTER

perform_print_screen

capture the screen as png and save to the rest results folder

perform_print_screen C://myFilePath

select_option

select an option from a select element

perform_select_option #element "Hello World"

switch_to_frame

set the focus on a defined frame element so that it becomes the queryable context

perform_switch_to_frame #element

switch_to_default

switch back to the default window, normally used to exit a frame

perform_switch_to_default

switch_to_tab

for the given index, switch to that tab

perform_switch_to_tab 0

drag_by

drag and element by x and y number of pixels.

perform_drag_by #element 77 108

hover_over_element

move the cursor over an element

perform_hover_over_element #element

mouse_drag

perform a click a drag operation on an element from a start location to a target location - like a box select

perform_mouse_drag "25 77" "88 100"

Assert Actions

attributes

validate multiple attributes checking that their values are as expected

assert_attributes #elem value1=myVal value2=yourVal

attribute_eq

Check that an attribute value and expected value match

assert_attribute_eq #element myAttr "true"

attribute_neq

Check that an attribute value and expected value do NOT match

assert_attribute_neq #element myAttr "true"

child_count_eq

check that the number of child elements match the expected count

assert_child_count_eq #element 5

child_count_neq

check that the number of child elements are NOT equal to the expected count

assert_child_count_neq #element 5

style_property_eq

check that an element's style property has an expected value

assert_style_property_eq #element background blue

style_property_neq

check that an element's style property is different from an expected value

assert_style_property_neq #element background pink

element_property_eq

check that an element's property has an expected value

assert_element_property_eq #my-input "hello world"

element_property_neq

check that an element's property has a different value from expected value

assert_element_property_neq #my-input "hello world"

tag_name_eq

check that an element tag name is what was expected

assert_tag_name_eq #element BUTTON

tag_name_neq

check if an element tag name is different from what was expected

assert_tag_name_neq #element BUTTON

text_content_eq

check that the text content of an element matches the defined value

assert_text_content_eq #element "Hello World"

text_content_neq

check that the text content of an element matches the defined value

assert_text_content_neq #elemebt "Goodbye World"

value_eq

check that the value property of an input matches the defined value

assert_value_eq #element 777

value_neq

check if the value property of an input does NOT match the defined value

assert_value_neq #element 777

element_exists

assert that an element is in the dom

assert_element_exists #element

element_not_exists

assert that an element is NOT in the dom

assert_element_not_exists #element

Wait Actions

time

wait for a given amount of time

wait_time 5

is_ready

wait until the defined element's dataset has a ready property set to true

wait_is_ready #element

element

wait until element exists and is visible

wait_element #element

attribute

wait for attribute to have a particular value

wait_attribute #element myAttr true

attributes

wait for multiple attributes to have their defined values

wait_attributes #element value1=1 value2=2

style_property

wait for element's style to have the defined property value

wait_style_property #element border none

element_properties

wait for element's style properties to be set to defined values

wait_element_properties #element type=text src=mySrc

text_content

wait until element's text content equals defined text

wait_text_content #element "Hello World"

text_value

wait until element's value property equals defined value

wait_text_value #element true

selected

wait for checkbox or radio buttons to be selected

wait_selected #checkbox

child_count

wait until defined element has a defined number of children

wait_child_count #container 7

element_count

wait until queried elements equal defined count

wait_element_count #container 7

window_count

wait until window count equals defined

wait_window_count #element 3

idle

wait for cpu idle

wait_idle

has_attribute

wait until an element has the defined attribute

wait_has_attribute #element status

has_not_attribute

wait until the defined attribute is no longer on element

wait_has_not_attribute #element status

System Actions

attributes_to_variables

create variables from element attribute values for future use

system_attributes_to_variables myVar val1=1 val2=2

properties_to_variables

create variables from element property values for future use

system_properties_to_variables myVar val1=1 val2=2

system_set_variables

Set variables in dictionary for future use

system_set_variables firstName=John lastName=Doe

set_uuid_variables

Generates uuid and assign them to variables

system_set_uuid_variables code1 code2

sleep

Let's the system sleep for defined duration

system_sleep 5
